# duilib-miniblink
duilib 接入miniblink 

1.最近项目要用到duilib+miniblink，于是将duilib项目下duilib多标签浏览器demo_csk中把ie改成miniblink做了个demo
  封装了个WkeWebKitUI控件，实现简单的js和c++相互调用
  
2.项目下载完成直接编译，将demo中的miniblink拷贝到生成目录下

3.要使用的话请遵守miniblink的相关协议
